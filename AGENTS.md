# Team Agent Public Repository

This repository is the canonical public development and release repository for Team Agent.

- Repository name: `team-agent-public`
- GitHub remote: `git@github.com:Florious95/team-agent.git`
- Build npm packages and publish GitHub updates from this repository only.
- Keep this repository public-safe: do not add `docs/`, `.team/`, `.claude/`, provider profile files, `.env` files, transcripts, private notes, tokens, keys, or local run artifacts.
- If older private workspaces contain useful code, copy only intentional public source changes into this repository, then test and publish from here.

Do not continue Team Agent public development in old private or experimental workspaces.
